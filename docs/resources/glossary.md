# Glossary

Common terms and definitions used in SolZ Bridge.

## Bridge Terms

**Bridge**
: A system that enables asset transfers between different blockchains.

**Custodial Bridge**
: A bridge where a trusted entity controls the locked assets.

**wZEC (Wrapped ZEC)**
: SPL token on Solana representing locked ZEC on Zcash blockchain.

**Reserve**
: Total amount of ZEC locked in the bridge.

**Outstanding wZEC**
: Total wZEC tokens minted minus tokens burned.

## Zcash Terms

**Shielded Address**
: A private Zcash address (z-address) that hides transaction details.

**Transparent Address**
: A public Zcash address (t-address) similar to Bitcoin addresses.

**Sapling**
: Current shielded transaction protocol in Zcash.

**Memo Field**
: Encrypted message field in Zcash shielded transactions.

**Confirmation**
: Number of blocks mined after a transaction.

## Solana Terms

**SPL Token**
: Token standard on Solana (like ERC-20 on Ethereum).

**Mint**
: Creating new tokens on Solana.

**Burn**
: Destroying tokens on Solana.

**Signature**
: Transaction ID on Solana.

**Devnet**
: Solana's test network for developers.

**Authority**
: Account with special permissions in a program.

## Technical Terms

**RPC (Remote Procedure Call)**
: Method for communicating with blockchain nodes.

**PDA (Program Derived Address)**
: Deterministic address in Solana programs.

**Anchor**
: Framework for building Solana programs.

**Testnet**
: Test network for Zcash.

**TXID**
: Transaction identifier (hash).
